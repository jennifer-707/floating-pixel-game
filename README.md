# Floating Pixel Game

## Descripción

Floating Pixel es un videojuego inspirado en la dinámica de Flappy Bird, desarrollado en Python utilizando Pygame.

El jugador debe controlar un personaje y evitar chocar con los tubos para conseguir la mayor puntuación posible. Además, el juego permite elegir entre varios personajes antes de comenzar la partida.

## Funciones del juego

* Menú principal.
* Selección de personajes.
* Sistema de puntuación.
* Récord máximo.
* Menú de pausa.
* Pantalla de Game Over.
* Música de fondo.
* Generación aleatoria de tubos.

## Archivos del proyecto

* **main.py:** ejecuta el juego y conecta todos los módulos.
* **config.py:** guarda constantes y configuraciones generales.
* **assets.py:** carga imágenes y recursos del juego.
* **player.py:** controla al personaje.
* **pipes.py:** genera y mueve los tubos.
* **menus.py:** contiene los diferentes menús.
* **game_logic.py:** maneja las colisiones y la puntuación.

## Tecnologías utilizadas

* Python
* Pygame
* Git
* GitHub

## Cómo ejecutar el juego

1. Instalar Python.
2. Instalar Pygame:

```bash
pip install pygame
```

3. Ejecutar:

```bash
python main.py
```
